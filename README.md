## laravel-svelte-starter

Starter template using Laravel and Svelte.

This starter template includes:
- Laravel 10
- Svelte
- Inertia.js
- Vite
- Tailwind CSS
- daisyUI

### Installation

Run following commands:

```
composer install
npm install
cp .env.example .env
php artisan key:generate
```

To try this app, run these commands on seperate terminal:

```
npm run dev
php artisan serve
```

### Deployment

Build the app:

```
npm run buid
```
